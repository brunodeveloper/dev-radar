# dev-radar
Semana OmniStack 10.0 - Rocketseat


## Prerequisites

* Node
* Express
* Yarn
* MongoDB

## .env Example

Add your connection MongoDB

```
MONGOOSE_CONNECT=YOUR-CONNECTION

```

## Backend

To start the server:

* Install dependencies with `yarn install`
* Run `yarn dev`

Now you can visit [`localhost:3333`](http://localhost:3333) from your browser.
